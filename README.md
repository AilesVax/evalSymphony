 Thème 
 ecole, classe, eleve
 ce projet vous permet de vous occuper de la gestion des élèves dans une école. vous pouvez ajouter des élèves dans une classe, modifier 
 les supprimer

 Installation du projet 
 cloner le projet : git clone https://github.com/AilesVax/evalSymphony
 installer composer : composer install
 creer la base de donnée : php bin/console doctrine:database:create
 executer les migrations : php bin/console doctrine:migrations:migrate
